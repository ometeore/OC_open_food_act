Pojet OpenFoodFact
==================


Ce programme à été fait dans le cadre d'un projet openClassroom par PILT.
Il est versionné sous gitHub: ***https://github.com/ometeore/OC_open_food_act.git***

installation
------------------
pour installer l'application il faut en installer python3 et les dépendances (je recommande l'utilisation d'un environnement virtuel)
To run this project on your computer, you need to install python and some dependencies (I suggest the use of a venv):

***pip install -r requirement.txt***

***source venv/bin/activate***

then to run the game:

***python main.py***

use 
--------

L'application a pour but de rechercher dans une base de donnée un aliment, d'éventuellemnt pouvoir le substituer et de sauvegarder le résultat de cette recherche. Les sujets associé à ce projet sont la gestion d'une base de donnée (localhost) et l'utilisation des api.

This app job is to search in a database an aliment, to eventually substitute him and save the result of this research. The topic link to this project are the use of a database and of an API (openfood fact).
